Detailed Description

A lightweight 3-component (x, y, z) float vector.

Ultraleap reports all spatial data (palm position, fingertip joints, palm velocity) in millimetres in the device coordinate frame. Vector3 wraps those triples and provides the small set of operations the gesture layer needs: subtraction (to form direction vectors between joints), length/distance, and the planar angle helpers used by the rotate gesture.
