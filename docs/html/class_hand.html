Detailed Description

Immutable snapshot of one hand for a single frame.

The Ultraleap SDK reports a hand as a raw LEAP_HAND C struct full of nested joints and bones. Hand copies out only the quantities the gesture layer actually uses — pinch/grab strengths, palm position and velocity, the five fingertip positions and their extended flags — into simple typed accessors. This keeps the rest of the codebase free of LeapC details.

A Hand is constructed once per frame from the SDK data and is read-only thereafter.
