Detailed Description

Per-hand memory carried between frames.

Gesture recognition is stateful: smoothing, debouncing, tap arming, swipe sustain counters, the zoom accumulator, the rotate baseline, drag bookkeeping and the pose-stability machine all need to remember the previous frame(s). One HandState exists per tracked hand id (see GestureDetector::states_).
