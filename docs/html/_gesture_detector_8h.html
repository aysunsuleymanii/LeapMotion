enum class Pose
strong

The coarse hand shape a frame is classified into.

Exactly one pose is active per hand at a time. Each pose owns a family of gestures, which keeps unrelated gestures from interfering (e.g. zoom lives on Pinch, rotate on OpenHand, so they can never fight each other).

Enumerator
None 

No fingers extended / hand open and idle — ignored.

Fist 

High grab strength → drag / drop.

OneFinger 

Index only → cursor movement + tap-to-click.

TwoFinger 

Index + middle → scroll / right-click / swipe / smart-zoom.

Pinch 

Thumb + index, changing distance → zoom.

OpenHand 

All fingers extended, turned like a dial → rotate.
